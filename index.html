About This Project


Project: Cloud-Based Audio-to-Text Transcription Service

Course: CS6620 - Cloud Computing

Group: Group 9


Features:

  • Upload audio files (MP3, WAV, M4A)
  • Automatic speech-to-text transcription using AWS Transcribe
  • Generate subtitle files (SRT, VTT, LRC formats)
  • Download transcripts and subtitle files
  • Track transcription job status in real-time

Architecture:

  • Frontend: Static HTML/JS hosted on S3
  • API: AWS API Gateway
  • Backend: AWS Lambda (Serverless)
  • Storage: AWS S3
  • Database: AWS DynamoDB
  • Transcription: AWS Transcribe
